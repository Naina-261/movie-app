# Movie Search App

## Description
This is a web application that allows users to search and explore movies using a public API. Users can view movie details and interact with the data dynamically.

## API Used
OMDB API (http://www.omdbapi.com/)

## Features
- Search movies by name
- Filter movies by type or year
- Sort movies alphabetically or by rating
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API

## How to Run
1. Clone the repository
2. Open index.html in your browser
